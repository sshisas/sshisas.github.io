About Me

I am a programmer, reverse engineer, and malware analyst. I have designed and engineered data science software and machine learning algorithms. I created software which translated data from the membrane inlet mass spectrometer on the Scripps Pier as LabVIEW data into Python plot.ly graphs which were hosted in iframes using Github Pages on ecoobs.ucsd.edu, a website I created. I also attended a reverse engineering training program at Google San Francisco called Blackhoodie. When it comes to reverse engineering, I have reversed and analyzed many malware samples and created static and dynamic analysis reports. Before attending university for cybersecurity, I worked in cancer and infectious disease research.
